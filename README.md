# Invoice Pro SA

A simple web-based invoice generator that allows you to create invoices, send them via WhatsApp, copy the text, or download as PDF.

## Features

- Add business name, customer name, and WhatsApp number
- Add multiple items with description, quantity, and price
- Generate invoice with total calculation
- Send invoice via WhatsApp
- Copy invoice text to clipboard
- Download invoice as PDF
- Save customer information locally

## Usage

1. Open `index.html` in a web browser
2. Fill in the business name, customer name, and WhatsApp number
3. Add items by clicking "+ Add Item"
4. Click "Generate Invoice" to create the invoice
5. Use the buttons to send via WhatsApp, copy text, or download PDF

## Technologies

- HTML5
- CSS3
- JavaScript
- html2pdf.js library for PDF generation
